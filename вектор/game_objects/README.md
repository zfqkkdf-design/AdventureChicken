# Игровые объекты

## player/
Загружайте сюда спрайты игрока:
- Анимации персонажа
- Состояния (стояние, прыжок, падение)

**Примеры файлов:**
- `player_stand.svg`
- `player_jump.svg`

## obstacles/
Загружайте сюда препятствия:
- Препятствия на земле (сено, камни)
- Препятствия в воздухе (птицы)

**Примеры файлов:**
- `hay_bale.svg`
- `stone.svg`
- `enemy_bird.svg`

## collectibles/
Загружайте сюда собираемые предметы:
- Яйца
- Семена
- Другие коллекционные предметы

**Примеры файлов:**
- `egg.svg`
- `seed.svg`
