## Как проверить, что ничего не сломалось:

### 1. Визуальная проверка в игре:
- Запустите игру
- Проверьте размеры солнца и луны
- Сравните с оригинальной версией

### 2. Если размеры отличаются:

#### Быстрый откат (вернуть PNG):
В файле `scenes/Game.tscn` замените обратно:
```
[ext_resource type="Texture2D" uid="uid://5n4g4qr1sm4u" path="res://assets/sunmoon/Group 822.png" id="8_sun"]
[ext_resource type="Texture2D" uid="uid://c56qolqldn6ce" path="res://assets/sunmoon/Group 830.png" id="9_moon"]
```

#### Настройка размера SVG (если нужно):

**Вариант A: Изменить размер в SVG файле**
Откройте `вектор/environment/sunmoon/sun.svg` и измените первую строку:
```xml
<svg width="НУЖНАЯ_ШИРИНА" height="НУЖНАЯ_ВЫСОТА" viewBox="0 0 НУЖНАЯ_ШИРИНА НУЖНАЯ_ВЫСОТА">
```

**Вариант B: Добавить масштаб в сцене**
В `scenes/Game.tscn` добавьте scale к Sprite2D:
```
[node name="Sun" type="Sprite2D" parent="BackgroundLayer/SunMoonLayer"]
position = Vector2(-100, 200)
scale = Vector2(МАСШТАБ_X, МАСШТАБ_Y)
texture = ExtResource("8_sun")
```

**Вариант C: Настроить масштаб импорта**
1. Выберите SVG файл в Godot
2. В Inspector найдите `svg/scale`
3. Измените значение (например, 0.5 для уменьшения в 2 раза)

### 3. Проверка других элементов:

Перед заменой других текстур:
1. ✅ Запишите размеры оригинального PNG
2. ✅ Проверьте размеры SVG
3. ✅ Если отличаются - настройте перед заменой
4. ✅ Сохраните оригинальные PNG до полной проверки
