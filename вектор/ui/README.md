# UI элементы

## buttons/
Загружайте сюда кнопки интерфейса:
- Кнопка паузы
- Кнопки меню (Play, Settings и т.д.)
- Кнопки паузы (Resume, Restart, Home)

**Примеры файлов:**
- `pause_button.svg`
- `resume_button.svg`
- `restart_button.svg`
- `home_button.svg`

## icons/
Загружайте сюда иконки:
- Иконка яйца
- Иконка семени
- Иконка сердечек (жизни)
- Другие UI иконки

**Примеры файлов:**
- `egg_icon.svg`
- `seed_icon.svg`
- `heart_icon.svg`

## controls/
Загружайте сюда элементы управления:
- Стрелки управления
- Джойстики
- Кнопки направления

**Примеры файлов:**
- `arrows.svg`
- `joystick.svg`
