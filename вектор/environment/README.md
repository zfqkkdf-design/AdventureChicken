# Окружение

## sunmoon/
Загружайте сюда солнце и луну:
- Солнце
- Луна
- Анимации смены дня/ночи

**Примеры файлов:**
- `sun.svg`
- `moon.svg`

## clouds/
Загружайте сюда облака:
- Различные типы облаков
- Анимации облаков

**Примеры файлов:**
- `cloud.svg`
- `cloud_small.svg`
- `cloud_large.svg`

## background/
Загружайте сюда фоновые элементы:
- Фоновые декорации
- Элементы заднего плана

**Примеры файлов:**
- `background_layer.svg`
- `decoration.svg`

## ground/
Загружайте сюда элементы земли:
- Дорога
- Трава
- Грунт

**Примеры файлов:**
- `road.svg`
- `grass.svg`
