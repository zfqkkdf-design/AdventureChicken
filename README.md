# Runner Template - Шаблон для раннер-игры

Шаблон для создания раннер-игры в стиле "Chicken Run" на базе Godot 4.5.

## Структура проекта

```
runner-template/
├── scenes/          # Сцены игры
│   ├── Menu.tscn    # Главное меню
│   ├── Game.tscn    # Игровая сцена
│   ├── Player.tscn  # Персонаж
│   ├── Obstacle.tscn # Препятствия
│   └── Collectible.tscn # Собираемые предметы
├── scripts/         # Скрипты
│   ├── Menu.gd      # Логика меню
│   ├── Game.gd      # Основная игровая логика
│   ├── Player.gd    # Управление персонажем
│   ├── Obstacle.gd  # Логика препятствий
│   ├── Collectible.gd # Логика собираемых предметов
│   └── GameManager.gd # Менеджер игры (автозагрузка)
└── project.godot    # Конфигурация проекта
```

## Основные возможности

- ✅ Главное меню с кнопками PLAY и SETTINGS
- ✅ Игровая сцена с персонажем
- ✅ Система прыжка (кнопка JUMP)
- ✅ Счетчик очков (яйца)
- ✅ Генерация препятствий и собираемых предметов
- ✅ Система паузы
- ✅ Сохранение лучшего результата
- ✅ Поддержка вертикальной и горизонтальной ориентации экрана

## Как использовать

1. Откройте проект в Godot 4.5
2. Запустите игру - откроется главное меню
3. Нажмите PLAY для начала игры
4. Используйте кнопку JUMP для прыжка персонажа
5. Собирайте яйца для увеличения счета
6. Избегайте препятствий

## Настройка

### Изменение скорости игры
В `scripts/Game.gd` измените:
- `spawn_interval` - интервал появления объектов
- В `scripts/Obstacle.gd` и `scripts/Collectible.gd` измените `SPEED` - скорость движения объектов

### Изменение физики прыжка
В `scripts/Player.gd` измените:
- `JUMP_VELOCITY` - высота прыжка
- `GRAVITY` - сила гравитации

### Добавление графики
Замените эмодзи в сценах на спрайты:
- В `Player.tscn` замените Label на Sprite2D
- В `Obstacle.tscn` и `Collectible.tscn` аналогично
