# Audio Files

## Структура папок

- `music/` - Фоновая музыка игры
- `sfx/` - Звуковые эффекты (sound effects)

## Необходимые файлы

### Музыка (`music/`):
- `bg_music.ogg` - Основная фоновая музыка (должна зацикливаться)

### Звуковые эффекты (`sfx/`):
- `jump.ogg` - Звук прыжка
- `duck.ogg` - Звук наклона/приседания
- `collect_egg.ogg` - Сбор яйца
- `collect_seed.ogg` - Сбор семечка
- `hit.ogg` - Столкновение с препятствием
- `game_over.ogg` - Game Over звук

## Где взять звуки?

См. файл `AUDIO_RESOURCES.md` в корне проекта для списка бесплатных ресурсов.

## Рекомендуемые форматы

- **OGG Vorbis** (предпочтительно) - лучший баланс качество/размер
- **WAV** - без сжатия, высокое качество, большой размер
- **MP3** - поддерживается, но OGG предпочтительнее
