# Инструкция по разделению текстуры камней

## Как использовать скрипт TextureSplitter.gd

1. **Поместите файл с камнями** в папку `assets/obstacle ground/` и назовите его `stones.png`
   - Файл должен содержать 9 камней, расположенных в сетке 3x3

2. **Откройте Godot Editor**

3. **Запустите скрипт** (два способа):

   **Способ 1: Через меню Project (рекомендуется)**
   - В верхней панели редактора найдите меню **Project**
   - Выберите **Project** → **Tools** → **TextureSplitter**
   - Скрипт запустится автоматически
   - Результаты будут в консоли (Output внизу экрана)

   **Способ 2: Через меню Tools (если доступно)**
   - В верхней панели найдите меню **Tools**
   - Выберите **Tools** → **Run Script**
   - Выберите файл `scripts/TextureSplitter.gd`
   - Нажмите кнопку **Run** (или **Запустить**)

4. **Проверьте результат**:
   - Скрипт создаст папку `assets/obstacle ground/stones/`
   - В ней будут сохранены 9 файлов: `stone_1.png`, `stone_2.png`, ..., `stone_9.png`
   - Проверьте консоль (Output) - там будут сообщения о процессе

5. **Если нужно изменить пути**:
   - Откройте `scripts/TextureSplitter.gd`
   - Измените переменные `texture_path` и `output_dir` на нужные пути

## Где найти меню?

- **Project** - находится в верхней панели редактора, обычно между "Editor" и "Debug"
- **Tools** - также в верхней панели, может быть не во всех версиях Godot 4.5

## Примечания

- Скрипт предполагает, что камни расположены в сетке 3x3 (3 колонки, 3 ряда)
- Если структура другая, измените переменные `cols` и `rows` в скрипте
- После разделения текстуры камни автоматически появятся в игре как препятствия
